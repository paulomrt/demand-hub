# DemandHub

> Plataforma de Planejamento de Demanda (Forecast) — Ecosistema StockHub

**DemandHub** é uma plataforma de planejamento de demanda que permite a criação, comparação e gestão de múltiplos planos de forecast em diferentes níveis de uma hierarquia comercial. A ferramenta faz parte do ecossistema StockHub, mantendo consistência visual total.

## 🚀 Principais Objetivos

- **Planejamento Multinível**: SKU × Hierarquia Comercial × Período.
- **Comparação de Planos**: Histórico, Orçamento, Forecast 1, Forecast 2, etc.
- **Inteligência**: Indicadores estatísticos e sugestões inteligentes de forecast.
- **Gestão Hierárquica**: Sistema de rateio hierárquico e de-para (mapeamento) de produtos e hierarquia.
- **Controle Granular**: Gestão de etapas, rodadas de planejamento e permissões.
- **Financeiro**: Gerenciamento de preços e detecção de lacunas.
- **Dashboard Modular**: KPIs customizáveis e internacionalização (PT-BR/EN).

## 🛠️ Stack Tecnológica

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **Gráficos**: [Recharts](https://recharts.org/)
- **Estado Global**: [Zustand](https://github.com/pmndrs/zustand)
- **Banco de Dados & Auth**: [Supabase](https://supabase.com/)
- **Validação**: [Zod](https://zod.dev/)
- **Data Fetching**: [TanStack React Query](https://tanstack.com/query/latest)
